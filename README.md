# 🐍 Neon Snake

A modern neon-styled Snake game built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just pure browser fun.

**[🎮 Play Now](https://snake-game-five-eta-41.vercel.app)**

## Features

- **Neon Aesthetics** — Gradient snake body with glowing effects
- **Particle Explosions** — Burst of particles when eating food
- **Screen Shake** — Satisfying feedback on each score
- **Persistent High Score** — Saved via localStorage
- **Responsive** — Adapts to mobile screens
- **Touch Support** — Swipe to control on mobile devices
- **Keyboard Support** — Arrow keys or WASD

## How to Play

1. Click **START** to begin
2. Use **Arrow Keys** or **WASD** to control the snake
3. On mobile, **swipe** in the direction you want to go
4. Eat the glowing food to grow and earn points
5. Don't run into walls or yourself!

## Tech Stack

- HTML5 Canvas
- Vanilla CSS (no preprocessor)
- Vanilla JavaScript (no framework)

## Run Locally

```bash
git clone https://github.com/ethanzhang666888/snake-game.git
cd snake-game
open index.html
```

No build step needed. Just open `index.html` in your browser.
